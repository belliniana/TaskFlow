# TaskFlow

Aplicação de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Concluir tarefas
- Pesquisa de tarefas
- Filtro por status
- Persistência com LocalStorage
- Contador de tarefas

## Conceitos Trabalhados

- DOM
- Eventos
- Arrays
- Objetos
- Filter
- LocalStorage
- JSON
- Manipulação dinâmica de elementos
